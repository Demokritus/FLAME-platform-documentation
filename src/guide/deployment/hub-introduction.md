## Requirements
The following guide is based on some shared assumptions:

- OS `debian` or `ubuntu`
- Nginx `v1.18.x` is installed
- Docker `v20.x` is installed
- Min. `4` cores
- Min. `20G` hard disk
- Domains with certificates
